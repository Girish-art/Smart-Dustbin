# Smart Dustbin Using Arduino
## Project Overview
This project demonstrates a Smart Dustbin using Arduino Uno and an ultrasonic sensor. When a person brings their hand near the dustbin, the lid automatically opens using a servo motor and closes after a few seconds.
## Components Used
- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- Servo Motor (SG90)
- Breadboard
- Jumper Wires
- USB Cable
